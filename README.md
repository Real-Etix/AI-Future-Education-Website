# Website

### Frontend

To build the Vue website,

```sh
cd frontend
npm install
npm run build
cd ..
```

To install the emoji-mart-vue,

```sh
npm install --save emoji-mart-vue-fast@15.0.4
```
```sh
please use chrome browser to enable voice input 
```
### Backend

To run the website with backend,

```sh
python backend/app.py
```
